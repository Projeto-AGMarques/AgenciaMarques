### CONECTE SUA CONTA!!
> git config --global user.name "Seu Nome"

Coloque o código acima e escreva o nome da sua conta do GitHubt.

> git config --global user.email "seuemail@example.com"

Coloque o código acima e escreva o nome do seu email usado no gitHub.

> git config --global --list

Verifica se seu nome e email foram atualizados.

___

### Como subir minha atualização no GitHub PELA PRIMEIRA VEZ?
#### 1. Primeiro Passo:
> git branch -r

Verifica quais são as branchs existentes no gitHub. ⬆

#### 2. Segundo passo:
> git checkout [nome_da_branch]

Entra na branch escolhida ⬆

#### 3. Terceiro passo:
> git add .

Salva suas alterações no git local da sua máquina (mais ou menos isso)

#### 4. Quarto passo:
> git commit -m "Escrever uma mensagem breve sobre sua atualização aqui"

Basicamente isso "compila", "prepara", o que você salvou na sua máquina para ser enviado para o GitHub.

#### 5. Quinto passo:
> git push origin [nome_da_sua_branch_atual]

Esse comando acima irá pegar o arquivo que foi preparado junto de um comentário e irá subir para sua branch escolhida.
